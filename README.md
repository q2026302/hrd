# 数字华容道 - 游戏介绍及说明

## 游戏简介

数字华容道是一款经典的益智滑动拼图游戏。玩家通过拖动或点击木块，利用空格移动数字，最终将数字按顺序排列。支持多种网格尺寸（默认3x3）。

## 游戏特点

### 🪵 木质玩具质感
- 立体木块设计，带有木纹纹理
- 木盘凹槽效果，空格处露出木板底色
- 点击时有轻微按压感

### 🖱️ 操作方式
- **拖动**：按住木块拖到相邻空格处松开
- **点击**：直接点击相邻木块也可移动

### 📊 辅助功能
- **移动记录**：实时记录每一步操作（如：5↑、3←）
- **计时与步数**：首次有效移动开始计时，步数同步累计
- **自动求解**：计算解法并显示步骤
- **步骤导航**：上一步/下一步逐步移动，点击步骤可直接跳转
- **可解性判断**：求解前自动判断是否有解

## 界面说明

### 左侧棋盘区
- **可切换网格**：支持多种尺寸（默认3x3）
- **数字木块**：顺序数字，带立体效果
- **空格**：凹槽样式，不可点击

### 右侧控制区
1. **控制面板**
   - **开局**（下拉菜单）
     - **重置棋盘**：回到当前开局的初始状态
     - **随机开局**：从目标状态合法随机步打乱（保证可解）
     - **简单开局**：较少步打乱（更易）
     - **困难开局**：更多步打乱（更难）
     - **手动设置**：弹窗输入自定义棋盘
   - **求解**：计算并显示解法步骤

2. **信息面板**
   - **移动记录**：显示已执行的所有移动
   - **求解步骤**：点击“求解”后显示步骤导航

## 游戏规则

1. **目标**：将数字按顺序排列，空格在右下角  
   ```
   目标状态（3x3）：
   ┌───┬───┬───┐
   │ 1 │ 2 │ 3 │
   ├───┼───┼───┤
   │ 4 │ 5 │ 6 │
   ├───┼───┼───┤
   │ 7 │ 8 │   │
   └───┴───┴───┘
   ```

2. **移动规则**
   - 只能移动与空格相邻的木块
   - 木块滑入空格位置
   - 空格本身不可移动

3. **可解性**
   - 手动输入可能无解，求解时会提示
   - 随机/简单/困难开局均由合法移动生成，保证可解

## 操作指南

### 基本移动
- **拖动**：按住木块拖到相邻空格
- **点击**：直接点击相邻木块

### 开局选项
点击“开局”下拉菜单：
- **重置棋盘**：回到当前开局的初始状态
- **随机开局**：可解随机布局
- **简单开局**：少量随机移动
- **困难开局**：较多随机移动
- **手动设置**：弹窗输入布局

### 手动输入
1. 点击“手动设置”打开弹窗  
2. 输入数字（空格留空）
3. 点击确认应用

### 自动求解
1. 点击“求解”计算解法
2. 步骤显示为“数字+方向”格式（如：5↑）
3. 使用上一步/下一步逐步移动
4. 点击任意步骤标签可跳转
5. 若不按求解步骤移动，求解结果会被清除

### 移动记录
- 右侧记录每一步操作
- 最新一步高亮显示

---

## 技巧提示

1. 先观察空格位置，再决定移动
2. 先固定边角数字，再处理中间
3. 求解功能可用于学习最优步骤
4. 若操作偏离求解步骤，系统会清除当前解法

## 常见问题

**Q：随机开局会出现无解吗？**  
A：不会。随机/简单/困难开局都是通过合法移动生成，保证可解。

**Q：箭头方向是什么意思？**  
A：表示数字移动方向，例如“5↑”表示数字5向上移动一格。

**Q：如何回到初始状态？**  
A：点击“重置棋盘”即可回到当前开局的初始布局。

**Q：手动输入时如何表示空格？**  
A：输入框留空即可表示空格（不支持输入0）。
