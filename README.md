# 数字华容道 - 游戏介绍及说明

## 游戏简介

数字华容道是一款经典的益智滑动拼图游戏。在一个3x3的网格中，放置着1-8的数字木块和一个空格。玩家需要通过拖动或点击木块，利用空格的位置移动数字，最终将数字按照1-8的顺序排列整齐。

## 游戏特点

### 🪵 木质玩具质感
- 立体的木块设计，带有木纹纹理
- 木盘凹槽效果，空格处露出木板底色
- 木块点击时有轻微的按压感

### 🖱️ 操作方式
- **拖动**：按住木块拖到相邻的空格处松开
- **点击**：直接点击与空格相邻的木块也可移动

### 📊 辅助功能
- **移动记录**：实时记录每一步操作（如：5↑、3←）
- **自动求解**：计算最优解法
- **步骤导航**：通过上一步/下一步查看解法，点击步骤可直接跳转
- **可解性判断**：自动检测当前棋盘是否有解

## 界面说明

### 左侧棋盘区
- **3x3网格**：9个格子，其中8个有数字木块，1个空格
- **数字木块**：1-8的数字，带有立体效果
- **空格**：深色凹槽，不可点击

### 右侧控制区
1. **控制面板**
   - **随机**：下拉菜单可选择"完全随机"、"简单开局"或"困难开局"
   - **手动**：打开弹窗手动输入棋盘布局
   - **求解**：计算并显示最优解法步骤
   - **重置**：回到初始状态

2. **信息面板**
   - **移动记录**：显示已经执行的所有移动步骤
   - **求解步骤**：显示程序计算出的解法步骤（点击求解后出现）

## 游戏规则

1. **目标**：将数字按照1-8的顺序排列，空格在右下角（第9格）

   ```
   目标状态：
   ┌───┬───┬───┐
   │ 1 │ 2 │ 3 │
   ├───┼───┼───┤
   │ 4 │ 5 │ 6 │
   ├───┼───┼───┤
   │ 7 │ 8 │   │
   └───┴───┴───┘
   ```

2. **移动规则**：
   - 只能移动与空格相邻的木块
   - 每次移动，木块滑入空格位置
   - 空格本身不可移动

3. **可解性**：
   - 并非所有随机排列都可解
   - 程序会自动判断当前棋盘是否可解
   - 无解时求解按钮会提示"当前棋盘无解"

## 操作指南

### 基本移动
- **拖动**：按住木块，拖到相邻的空格处松开
- **点击**：直接点击与空格相邻的木块

### 随机开局
点击"随机"按钮，从下拉菜单中选择：
- **完全随机**：完全随机打乱（150步随机移动）
- **简单开局**：只交换7和8的位置
- **困难开局**：预设的复杂布局

### 手动输入
1. 点击"手动"按钮打开输入弹窗
2. 在9个格子中输入数字1-8（空格留空或输入0）
3. 点击"应用"确认

### 自动求解
1. 点击"求解"按钮，程序计算最优解法
2. 求解步骤显示为"数字+箭头"格式（如：5↑ 表示数字5向上移动）
3. 使用"上一步/下一步"按钮逐步查看
4. 点击任意步骤标签可直接跳转到该步骤

### 移动记录
- 右侧上方显示每一步操作记录
- 最新一步高亮显示
- 记录格式：数字+方向箭头

## 技巧提示

1. **观察空格位置**：每次移动前先找到空格，思考哪个数字可以移入空格
2. **从边缘开始**：先固定边角的数字（如1、3、7、9），再处理中间
3. **利用求解功能学习**：卡顿时可点击求解学习最优解法
4. **自由探索**：如果不按求解步骤移动，求解结果会自动清除

## 常见问题

**Q：为什么有些随机布局无解？**
A：华容道游戏有奇偶性限制，大约一半的随机排列不可解。程序会自动检测并提示。

**Q：求解步骤中的箭头方向是什么意思？**
A：箭头表示数字的移动方向，例如"5↑"表示数字5向上移动一格。

**Q：如何回到初始状态？**
A：点击"重置"按钮即可回到最初的状态（随机开局后的状态）。

**Q：手动输入时如何表示空格？**
A：输入框留空，或者输入0均可表示空格。
